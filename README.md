# RTS Prototype

![Python](https://img.shields.io/badge/python-3.14-blue.svg)
![Arcade](https://img.shields.io/badge/arcade-3.0.0-green.svg)
![License](https://img.shields.io/badge/license-MIT-blue.svg)

Прототип RTS-стратегии на Python с использованием библиотеки Arcade.

## 📋 Возможности

- ✅ Управление юнитами (выделение, движение, атака)
- ✅ Система команд (ЛКМ - выделение, ПКМ - приказ)
- ✅ FSM (конечный автомат) для поведения юнитов
- ✅ Простой ИИ (автоатака ближайшего врага)
- ✅ Карта на основе Tiled (TMX)
- ✅ Система проходимости тайлов
- ✅ HP бары и визуализация состояний

## 🎮 Управление

- **ЛКМ** - выделить юнита
- **ПКМ** - приказать движение/атаку
- **ПКМ по врагу** - атаковать цель

## 🚀 Установка и запуск

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Запуск игры

```bash
cd src
python main.py
```

## 📁 Структура проекта

```
RTS_Prototype/
├── assets/                  # Все ресурсы игры
│   ├── images/              # Графика
│   │   ├── units/           # Спрайты юнитов
│   │   └── gfx/             # Тайлсеты
│   ├── sounds/              # Аудио
│   │   ├── music/           # Музыка
│   │   └── effects/         # Звуковые эффекты
│   └── maps/                # Карты Tiled
├── src/                     # Исходный код
│   ├── main.py              # Точка входа
│   ├── game.py              # Главный игровой класс
│   ├── config.py            # Константы и настройки
│   ├──/                     # Игровая логика и данные
│   │   ├── game_state.py    # Состояние игры
│   │   ├── game_map.py      # Логика карты
│   │   └── unit.py          # Модель юнита (FSM)
│   ├──/                     # Отрисовка и UI
│   │   ├── game_view.py     # Главное игровое окно
│   │   ├── menu_view.py     # Главное меню
│   │   └── unit_sprite.py   # Спрайты юнитов
│   └── input_controller.py  # Обработка ввода
├── .gitignore
├── requirements.txt
└── README.md
```

## 📂Архитектура (MVC)

- **Models** (`src/models/`) - игровая логика, данные, FSM
- **Views** (`src/views/`) - отрисовка, UI, анимации
- **Controllers** (`src/controllers/`) - обработка ввода
- **Game** (`src/game.py`) - центральный класс управления

## 🔧 Технологии

- **Python 3.14**
- **Arcade 3.3.3** - игровой движок
- **Pymunk** - физический движок
- **Tiled** - редактор карт
